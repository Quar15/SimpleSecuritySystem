POST /api/system/authorize

Example request body:

{ "systemId": "aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee", "cardValue": "TESTCARDX" }
  • systemId - uuid of the system
  • cardValue - value of the RFID card

Responses:

  • 400 - Bad Request
  • 404 - System with that id not found
  • 401 - Card is not authorized
  • 202 - Card is authorized
POST /api/event

Example request body:

{ "systemId": "aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee", "timestamp": 1705762594 }
  • systemId - uuid of the system
  • timestamp - event timestamp

Responses:

  • 400 - Bad Request
  • 404 - System with that id not found
  • 201 - Event saved
